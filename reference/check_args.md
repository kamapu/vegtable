# Check validity of arguments for statistics

Check validity of arguments in functions
[`trait_stats()`](http://kamapu.github.io/vegtable/reference/trait_stats.md)
and
[`trait_proportion()`](http://kamapu.github.io/vegtable/reference/trait_stats.md).

## Usage

``` r
check_args(
  object,
  trait,
  head_var,
  weight,
  taxon_levels,
  merge_to,
  trait_levels
)
```
