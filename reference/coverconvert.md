# Cover conversion tables

Cover conversion tables for
[vegtable](http://kamapu.github.io/vegtable/reference/vegtable-class.md)
objects.

This class implements conversions from different cover scales in
percentage cover. For transformations to percentage cover, the function
[`cover_trans()`](http://kamapu.github.io/vegtable/reference/cover_trans.md)
should be than used.

## Slots

- `value`:

  List containing the levels of each scale.

- `conversion`:

  List with the respective start and end cut levels for the scale
  levels.
