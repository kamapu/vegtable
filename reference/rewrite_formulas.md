# Re-writing formulas for print output

Mediating between syntax and print format.

## Usage

``` r
rewrite_formulas(shaker, companion)
```

## Arguments

- shaker:

  A [shaker](http://kamapu.github.io/vegtable/reference/shaker-class.md)
  object.

- companion:

  A companion data set which is either missing or a
  [vegtable](http://kamapu.github.io/vegtable/reference/vegtable-class.md)
  object.

## Value

A formated output text.
