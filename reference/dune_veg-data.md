# Dutch dune meadows as vegtable

Data set from the package
[vegan::vegan](https://vegandevs.github.io/vegan/reference/vegan-package.html),
converted to a
[vegtable](http://kamapu.github.io/vegtable/reference/vegtable-class.md)
object.

## Usage

``` r
dune_veg
```

## Format

An object of class
[`vegtable`](http://kamapu.github.io/vegtable/reference/vegtable-class.md).

## Source

Original data were imported from
[vegan::dune](https://vegandevs.github.io/vegan/reference/dune.html).

## References

**Jongman RHG, ter Braak CJF, van Tongeren OFR (1987).** *Data analysis
in community and landscape ecology.* Pudoc, Wageningen, NL.
