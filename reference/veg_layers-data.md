# Information on vegetation layers

A data frame with a standard classification on vegetation layers.

## Usage

``` r
veg_layers
```

## Format

A data frame.

## Examples

``` r
summary(veg_layers)
#>      layer   abbreviation       description       
#>  0      :1   Length:10          Length:10         
#>  1      :1   Class :character   Class :character  
#>  2      :1   Mode  :character   Mode  :character  
#>  3      :1                                        
#>  4      :1                                        
#>  5      :1                                        
#>  (Other):4                                        
```
